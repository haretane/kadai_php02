## ⑤工夫した点・こだわった点

スマホネイティブで、検索や評価、アップロードがしやすいようにUXを工夫しています。<br>
マイページや、そのお菓子の遍歴（パッケージの移り変わりや評価の推移）もつくろうとしているのですが、時間切れとなりました。<br>
ですが、見据えてデータをとっておかないと、後で「過去のデータを記録する」ことは不可能なので、必要だろう、というカラムは作成しました。
